# WebuntisNoten2Atlantis

Mit WebuntisNoten2Atlantis können die Zeugnisnoten von Webuntis nach Atlantis übertragen werden.

## Voraussetzungen

* Fächer- und Klassenbezeichnungen sind identisch in Untis und Atlantis.
* Notenblatt und Zeugnisformular wurden in Atlantis angelegt.
* Eine Prüfungsart wurde in Webuntis angelegt, die denselben Namen trägt, wie das Zeugnisformular in Atlantis.
* Der Benutzer hat Berechtigung Prüfungen aus Webuntis zu exportieren und SQL-Dateien in Atlantis zu importeieren .               

## Vorgehen

1. Prüfung aus Webntis exportieren und auf den Desktop legen.
2. webuntisnoten2atlantis starten.
3. Gewünschte Klassen filtern oder alle Klassen wählen.
4. Erzeugte SQL-Datei in Atlantis importieren.
